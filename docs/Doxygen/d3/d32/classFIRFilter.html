Detailed Description

template<uint8_t N, class T = float>
class FIRFilter< N, T >

Finite Impulse Response filter implementation.

Implements the following difference equation:

\[ y[n] = \sum_{i=0}^{N-1} b_i \cdot x[n-i] \]
