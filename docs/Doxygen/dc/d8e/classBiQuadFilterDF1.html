Detailed Description

template<class T = float>
class BiQuadFilterDF1< T >

Generic BiQuad (Bi-Quadratic) filter class, Direct Form 1 implementation.

Uses the NormalizingBiQuadFilterDF1 implementation for floating point types, and NonNormalizingBiQuadFilterDF1 for all other types.

Implements the following difference equation:

\[ y[n] = \frac{b_0 \cdot x[n] + b_1 \cdot x[n-1] + b_2 \cdot x[n-2] - a_1 \cdot y[n-1] - a_2 \cdot y[n-2]}{a_0} \]

\[ H(z) = \frac{b_0 + b_1 z^{-1} + b_2 z ^{-2}} {a_0 + a_1 z^{-1} + a_2 z ^{-2}} \]
